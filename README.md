# Homebrew Tap

Custom Homebrew formulae for my CLI tools.

## Installation

```bash
brew tap wdm0006/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [vanity](https://github.com/wdm0006/vanity) | Sync your GitHub contribution graph across multiple accounts |

## Usage

```bash
# Install a formula
brew install wdm0006/tap/vanity

# Or after tapping
brew tap wdm0006/tap
brew install vanity
```

## Updating

Formulae are automatically updated when new releases are published via GoReleaser.

```bash
brew update
brew upgrade vanity
```
